# Flexbox

* Nos permite positionar os elementos dentro da caixa.
* Controle em uma dimensão (horizontal ou vertical)
* Alinhamento, direcionamento, ordenar e tamanhos

```css
div.parent {
    display: flex;
}
```

## flex-direction

* Qual a direção do flex: horizontal ou vertial
* row | column

## alinhamento

* justify-content
* align-items
